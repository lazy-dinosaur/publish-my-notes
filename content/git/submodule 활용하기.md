# 다른 git 저장소를 내 프로젝트에 포함시키자!!

프로젝트를 진행하다 보면 이미 개발된 다른 프로젝트를 활용하여 진행해야 하는 경우가 있다. 자바스크립트나 파잇선을 활용한다면 npm이나 pip를 통해 다른 프로젝트를 가져와서 사용할 수 있지만, git 저장소를 활용하는 경우에는 submodule을 활용하여 다른 저장소를 내 프로젝트에 포함시킬 수 있다.

# 왜 필요했나?

지금까지 여러가지 토이프로젝트를 진행하면서 submodule 이 필요했던 적이 없었다. 하지만 최근 스플릿 키보드의 늪에 빠져버린 터라. 키보드의 펌웨어를 설정하고 키를 맵핑하는 작업을 하는 도중 아직 개발중인 기능들을 적용하기 위해서는 다른 저장소를 활용해야 했다. 그래서 submodule을 활용해보기로 했다.

## 매우 쉬운 submodule 활용

```bash
git submodule # 기본
```
